# KillStreak
Spigot plugin for counting kills of players.

----

## Dependencies
- PLJRApi: https://www.github.com/ThePlay3r/PLJRApi

## Features
- Customizable GUI
- Leaderboard
- Cross-Server
- MySQL Support
- BungeeCord Support
- Huge Configuration
